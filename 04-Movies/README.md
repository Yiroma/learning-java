# **Challenge**

## **Créer la filmographie d'Indiana**

1. Dans la méthode main d'une classe `Movies.java`, crée :

un premier tableau contenant 3 titres de films dans lesquels joue notre ami Indiana ([**Hint**](https://www.allocine.fr/rechercher/?q=indiana+jones)),un second tableau contenant, pour chaque film que tu as choisi, les 3 principaux acteurs (*Hint :* pense aux tableaux à plusieurs dimensions).

1. Une fois les tableaux créés, réalise une boucle pour afficher la liste des titres de films, ainsi que les acteurs associés de la manière suivante :Dans le film *title*, les principaux acteurs sont : *actor1*, *actor2*, *actor3*
2. La liste devra s'afficher à la compilation, puis exécution du code dans le terminal.
3. Pour soumettre tes solutions, uploade le fichier dans un gist, puis poste le lien de ce dernier.

## **Critères de validation**

- Le fichier contient les tableaux, et on y retrouve bien 3 films, ainsi que 3 acteurs par film.
- Une boucle est utilisée pour afficher les films et leurs acteurs.
- Lorsque tu exécutes le script depuis ton terminal, le résultat affiché est bien : "Dans le film *title*, les principaux acteurs sont : *actor1*, *actor2*, *actor3*".
