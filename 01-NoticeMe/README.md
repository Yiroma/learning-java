# **Challenge**

## **Crée ton propre script**

1. Crée un nouveau dossier dans `JavaProjects`, que tu nommeras `NoticeMe`.
2. Dans le dossier `NoticeMe`, crée la classe `Senpai.java`.
3. Déclare ta classe `Senpai` et ajoute une méthode *main()*, telle qu'elle est montrée dans la quête.
4. Dans ta méthode *main()*, ajoute le code permettant d'afficher le texte "Notice me Senpai".
5. Compile et exécute le code afin de tester qu'il fonctionne correctement.
6. Copie ton code dans un **Gist** et partage le fichier `Senpai.java` en solution.

## **Critères de validation**

- Le fichier doit bien s'appeler `Senpai.java` et la classe `Senpai`.
- La classe doit se compiler sans message d'erreur.
- À l'exécution de la classe compilée, le message "Notice me Senpai" doit apparaître dans le terminal.
